# 챕터 7 — 시장에서 (ბაზარში)

## 시장·쇼핑
ბაზარი = 시장 (market)
გამყიდველი = 판매원 (seller)
მყიდველი = 구매자, 손님 (buyer)
ფასი = 가격 (price)
ბოლო ფასი = 최종 가격 (final price)
ხურდა = 잔돈 (change)
კილო = 킬로그램 (kilo)
ნახევარი = 절반 (half)
ძვირი = 비싼 (expensive)
იაფი = 싼 (cheap)
რამე = 무언가 (something)
ვაჭრობა = 흥정, 거래 (bargaining)

## 과일 (ხილი)
ხილი = 과일 (fruit)
ვაშლი = 사과 (apple)
მსხალი = 배 (pear)
ატამი = 복숭아 (peach)
ქლიავი = 자두 (plum)
მარწყვი = 딸기 (strawberry)
ყურძენი = 포도 (grapes)
ბალი = 체리 (cherry)
ალუბალი = 사워 체리 (sour cherry)
ჟოლო = 라즈베리 (raspberry)
ბროწეული = 석류 (pomegranate)
გარგარი = 살구 (apricot)
კარალიოკი = 감 (persimmon)
ფორთოხალი = 오렌지 (orange)
მანდარინი = 귤 (tangerine)
ლიმონი = 레몬 (lemon)
გრეიპფრუტი = 자몽 (grapefruit)
კივი = 키위 (kiwi)
ბანანი = 바나나 (banana)
ანანასი = 파인애플 (pineapple)
ნესვი = 멜론 (melon)
საზამთრო = 수박 (watermelon)
ფინიკი = 대추야자 (date)

## 채소 (ბოსტნეული)
ბოსტნეული = 채소 (vegetables)
კარტოფილი = 감자 (potato)
კიტრი = 오이 (cucumber)
პამიდორი = 토마토 (tomato)
ხახვი = 양파 (onion)
ნიორი = 마늘 (garlic)
სტაფილო = 당근 (carrot)
კომბოსტო = 양배추 (cabbage)
ბადრიჯანი = 가지 (eggplant)
წიწაკა = 고추, 피망 (pepper)
ჭარხალი = 비트 (beet)
ბოლოკი = 무, 래디시 (radish)
ისპანახი = 시금치 (spinach)
მწვანილი = 허브, 푸른 채소 (herbs/greens)

## 고기 (ხორცი)
ხორცი = 고기 (meat)
თევზი = 생선 (fish)
ძროხა = 소 (cow)
ძროხის ხორცი = 쇠고기 (beef)
ღორი = 돼지 (pig)
ღორის ხორცი = 돼지고기 (pork)
ქათამი = 닭 (chicken)
ქათმის ხორცი = 닭고기 (chicken meat)
ინდაური = 칠면조 (turkey)
ცხვარი = 양 (sheep)
ცხვრის ხორცი = 양고기 (mutton)
ბატკანი = 어린 양 (lamb)
ბატკნის ხორცი = 어린 양고기 (lamb meat)
თხა = 염소 (goat)
ხბო = 송아지 (calf)
ხბოს ხორცი = 송아지고기 (veal)

## 동사 — 사다 (ვიყიდო)
მინდა ვიყიდო = 나는 사고 싶다 (I want to buy)
გინდა იყიდო = 너는 사고 싶다 (you want to buy)
უნდა იყიდოს = 그/그녀는 사고 싶어 한다 (he/she wants to buy)
გვინდა ვიყიდოთ = 우리는 사고 싶다 (we want to buy)
გინდათ იყიდოთ = 당신은 사고 싶다 (you want to buy)
უნდათ იყიდონ = 그들은 사고 싶어 한다 (they want to buy)
ყიდულობს = (그/그녀는) 산다 (buys)
იხდის = (그/그녀는) 지불한다 (pays)

## 시장 표현
რა ღირს? = 얼마예요? (How much is it?)
ბოლო ფასი? = 최종 가격은요? (Final price?)
ძალიან ძვირია = 너무 비싸요 (Too expensive)
აიღეთ! = 받으세요! (Take it / Here you are)
გინდათ კიდევ რამე? = 더 필요하세요? (Want anything else?)
მეტი არაფერი = 더는 없어요 (Nothing more)
რამდენი კილო? = 몇 킬로요? (How many kilos?)

## 시간 — 정시·오전/오후
რომელი საათია? = 몇 시입니까? (What time is it?)
საათი = 시, 시계 (hour/clock)
პირველი საათია = 1시입니다 (it's one o'clock)
ორი საათია = 2시입니다 (it's two o'clock)
თორმეტი საათია = 12시입니다 (it's twelve o'clock)
შუადღე = 정오, 한낮 (noon)
შუადღეა = 정오입니다 (it's noon)
შუაღამე = 자정, 한밤 (midnight)
შუაღამეა = 자정입니다 (it's midnight)
დილის = 오전의 (in the morning / AM)
შუადღის = 오후의, 낮의 (in the afternoon)
დღის = 낮의 (of the day)
საღამოს = 저녁의 (in the evening)
ღამის = 밤의 (at night)
დილის ცხრა საათი = 오전 9시 (9 AM)
შუადღის ორი საათი = 오후 2시 (2 PM)
საღამოს რვა საათი = 저녁 8시 (8 PM)
ღამის თერთმეტი საათი = 밤 11시 (11 PM)

## 시간 — 분·15분·반·정시 전
წუთი = 분 (minute)
მეოთხედი = 15분, 4분의 1 (quarter)
ნახევარი = 30분, 절반 (half)
აკლია = 모자라다 (lacks)
სამი საათი და ხუთი წუთი = 3시 5분 (3:05)
სამი საათი და თხუთმეტი წუთი = 3시 15분 (3:15)
სამი საათი და მეოთხედი = 3시 15분 (3:15, "그리고 1/4")
სამი საათი და ოცი წუთი = 3시 20분 (3:20)
პირველის ნახევარი = 12시 30분 ("1시의 절반")
სამის ნახევარი = 2시 30분 (2:30)
რვის ნახევარი = 7시 30분 (7:30)
ოთხ საათს თხუთმეტი წუთი აკლია = 3시 45분 ("4시에 15분 모자라다")
ოთხ საათს მეოთხედი აკლია = 3시 45분 ("4시에 1/4 모자라다")
ცხრა საათს ხუთი წუთი აკლია = 8시 55분 (8:55)
პირველის ნახევარზე = 12시 30분에 (at 12:30)
