# 챕터 8 — 나의 가족 (ჩემი ოჯახი)

## 가족 (ოჯახი)
ოჯახი = 가족 (family)
დედა = 어머니 (mother)
მამა = 아버지 (father)
მშობელი = 부모 (parent)
მშობლები = 부모님 (parents)
დედ-მამა = 부모 (mom and dad)
შვილი = 자녀, 아이 (child)
ქალიშვილი = 딸 (daughter)
ვაჟიშვილი = 아들 (son)
და = 자매, 누나, 여동생 (sister)
ძმა = 형제, 형, 남동생 (brother)
და-ძმა = 형제자매 (siblings)
ცოლი = 아내 (wife)
ქმარი = 남편 (husband)
ცოლ-შვილი = 처자식 (wife and children)
ქმარ-შვილი = 남편과 자식 (husband and children)

## 친척 (ნათესავები)
ბებია = 할머니 (grandmother)
ბაბუა = 할아버지 (grandfather)
პაპა = 할아버지 (grandpa)
შვილიშვილი = 손주 (grandchild)
ბიძა = 삼촌 (아버지의 형제) (uncle)
მამიდა = 고모 (아버지의 자매) (paternal aunt)
დეიდა = 이모 (어머니의 자매) (maternal aunt)
ბიძაშვილი = 사촌 (삼촌의 자녀) (cousin)
მამიდაშვილი = 고종사촌 (cousin, paternal aunt's child)
დეიდაშვილი = 이종사촌 (cousin, maternal aunt's child)
ნათესავი = 친척 (relative)
ნათესავები = 친척들 (relatives)
დედისერთა = 외동 (only child)

## 소유 합성 (가족 + 소유)
დედაჩემი = 나의 어머니 (my mother)
მამაჩემი = 나의 아버지 (my father)
მამაშენი = 너의 아버지 (your father)
ბიძაჩემი = 나의 삼촌 (my uncle)
მამიდაჩემი = 나의 고모 (my paternal aunt)
დეიდაჩემი = 나의 이모 (my maternal aunt)
დეიდამისი = 그/그녀의 이모 (his/her aunt)

## 동사 — 있다 (ჰყავს, 사람·동물)
მყავს = 나는 (~가) 있다 (I have)
გყავს = 너는 (~가) 있다 (you have)
ჰყავს = 그/그녀는 (~가) 있다 (he/she has)
გვყავს = 우리는 (~가) 있다 (we have)
გყავთ = 당신(들)은 (~가) 있다 (you have)
ჰყავთ = 그들은 (~가) 있다 (they have)
ჰქვია = (이름이) ~이다 (is named)

## 형용사·접속사
ბედნიერი = 행복한 (happy)
რადგან = 왜냐하면 (because)
ახალგაზრდა = 젊은 (young)
მოხუცი = 늙은 (old, elderly)
ლამაზი = 아름다운 (beautiful)

## 격조사 -თან / -სთან (~와, ~의 곁에)
ლევანთან = 레반과 / 레반네 (with/at Levan's)
ბანკთან = 은행 곁에 (near the bank)
ნინოსთან = 니노와 / 니노네 (with/at Nino's)
კაფესთან = 카페 곁에 (near the cafe)
ბებიასთან = 할머니와 / 할머니 댁에 (at grandma's)

## 복수형 -ები
შვილები = 자녀들 (children)
დები = 자매들 (sisters)
ძმები = 형제들 (brothers)
დედები = 어머니들 (mothers)
ბებიები = 할머니들 (grandmothers)
ნათესავები = 친척들 (relatives)
მანქანები = 자동차들 (cars)
