# 챕터 6 — 나의 사무실에서 (ჩემს ოფისში)

## 사무실·사물
ოფისი = 사무실 (office)
ბანკი = 은행 (bank)
საელჩო = 대사관 (embassy)
მაგიდა = 책상, 테이블 (desk/table)
სკამი = 의자 (chair)
საათი = 시계, 시간 (clock/hour)
კომპიუტერი = 컴퓨터 (computer)
ტელეფონი = 전화기 (telephone)
ნომერი = 번호 (number)
კედელი = 벽 (wall)
რუკა = 지도 (map)
ოთახი = 방 (room)
მიღება = 리셉션, 접수처 (reception)

## 사람·직장
ასისტენტი = 조수, 비서 (assistant)
კოლეგა = 동료 (colleague)
ბიზნესმენი = 사업가 (businessman)
შეხვედრა = 회의, 만남 (meeting)
კონფერენცია = 컨퍼런스 (conference)
მუშა = 노동자 (worker)

## 형용사·부사
ადრე = 일찍 (early)
ყოველთვის = 항상 (always)
დაკავებული = 바쁜 (busy)
პუნქტუალური = 시간을 잘 지키는 (punctual)
ოფიციალური = 공식적인 (official)
ასეთი = 그런, 이와 같은 (such)
სხვა = 다른 (other)
მთელი = 전체의, 모든 (whole)
ნათელი = 밝은 (light/bright)
ბნელი = 어두운 (dark)
საინტერესო = 흥미로운 (interesting)
ალბათ = 아마도 (probably)

## 부정·의문 표현
არავინ = 아무도 ~않다 (nobody)
არსად = 어디에도 ~않다 (nowhere)
არაფერი = 아무것도 ~않다 (nothing)

## 동사 — 일하다 (მუშაობს)
ვმუშაობ = 나는 일한다 (I work)
მუშაობ = 너는 일한다 (you work)
მუშაობს = 그/그녀는 일한다 (he/she works)
ვმუშაობთ = 우리는 일한다 (we work)
მუშაობთ = 당신은/당신들은 일한다 (you work)
მუშაობენ = 그들은 일한다 (they work)

## 요일 (კვირის დღეები)
ორშაბათი = 월요일 (Monday)
სამშაბათი = 화요일 (Tuesday)
ოთხშაბათი = 수요일 (Wednesday)
ხუთშაბათი = 목요일 (Thursday)
პარასკევი = 금요일 (Friday)
შაბათი = 토요일 (Saturday)
კვირა = 일요일 (Sunday)
შაბათ-კვირა = 주말 (weekend)

## 요일에 (-ს, ~요일에)
ორშაბათს = 월요일에 (on Monday)
პარასკევს = 금요일에 (on Friday)
შაბათს = 토요일에 (on Saturday)
კვირას = 일요일에 (on Sunday)

## 숫자 11~19
თერთმეტი = 11 (eleven)
თორმეტი = 12 (twelve)
ცამეტი = 13 (thirteen)
თოთხმეტი = 14 (fourteen)
თხუთმეტი = 15 (fifteen)
თექვსმეტი = 16 (sixteen)
ჩვიდმეტი = 17 (seventeen)
თვრამეტი = 18 (eighteen)
ცხრამეტი = 19 (nineteen)

## 숫자 20~100 (20진법)
ოცი = 20 (twenty)
ოცდაათი = 30 (thirty / 20+10)
ორმოცი = 40 (forty / 2×20)
ორმოცდაათი = 50 (fifty / 40+10)
სამოცი = 60 (sixty / 3×20)
სამოცდაათი = 70 (seventy / 60+10)
ოთხმოცი = 80 (eighty / 4×20)
ოთხმოცდაათი = 90 (ninety / 80+10)
ასი = 100 (hundred)

## 전화 표현
გისმენთ! = 여보세요! / 듣고 있습니다 (Hello! / I'm listening)
ალო! = 여보세요! (Hello! - on the phone)
თუ შეიძლება, სთხოვეთ = (전화) 바꿔 주시겠어요? (May I speak with ~?)
სხვა ნომერია = 다른 번호입니다 / 잘못 거셨습니다 (Wrong number)
სხვაგან მოხვდით = 잘못 거셨습니다 (Wrong number)
ნომერი შეგეშალათ = 번호를 잘못 아셨습니다 (You have the wrong number)
ბატონო? = 네? / 뭐라고 하셨죠? (Pardon? / What did you say?)
არაფრის! = 천만에요! (You're welcome!)

## 사무실 표현
დილიდან მუშაობს = 아침부터 일합니다 (works from the morning)
ყოველთვის ადრე მიდის = 항상 일찍 갑니다 (always goes early)
შეხვედრაზე მიდის = 회의에 갑니다 (goes to the meeting)
ოფისში არ იქნება = 사무실에 없을 것입니다 (will not be in the office)
კედელზეა = 벽에 있다 (is on the wall)
მაგიდაზეა = 책상 위에 있다 (is on the desk)
