# 챕터 3 — 소개 (გაცნობა)

## 인사·만남
გაცნობა = 소개, 지인, 만남
სასიამოვნო = 즐거운, 기분 좋은
სასიამოვნოა = 반갑습니다
ჩემთვისაც = 저도요 / 저도 반갑습니다
მშვიდობა = 평화
დილა მშვიდობისა = 좋은 아침입니다
საღამო მშვიდობისა = 좋은 저녁입니다
ღამე მშვიდობისა = 안녕히 주무세요 / 좋은 밤

## 시간
წამი = 초 (second)
წუთი = 분 (minute)
საათი = 시간, 시계 (hour / clock)
დღე = 일, 하루 (day)
დღეს = 오늘 (today)
დილა = 아침 (morning)
დილას / დილით = 아침에 (in the morning)
შუადღე = 오후 (afternoon)
შუადღეს = 오후에 (in the afternoon)
საღამო = 저녁 (evening)
საღამოს = 저녁에 (in the evening)
ღამე = 밤 (night)
ღამით = 밤에 (at night)
კვირა = 주 / 일요일 (week / Sunday)
კვირას = 일요일에 (on Sunday)
შუა = 중간 (middle)
თვე = 월, 달 (month)
წელი / წელიწადი = 년, 해 (year)

## 인물·사물·기타
პიროვნება = 성격, 인물 (person / personality)
პირი = 사람 (person)
ნივთი = 사물, 물건 (thing / object)
ენა = 언어 (language)
ეროვნება = 국적, 민족 (nationality)
თუ = 혹은 / 만약 (or / if)
სხვა = 다른 (other)
ძველი = 오래된 (old)

## 핵심 동사
ჰქვია = 이름이 ~이다 (His/her name is~)
მე მქვია = 내 이름은 ~이다 (My name is~)
ცოდნა = 알다 (to know)
მე ვიცი = 나는 (언어를) 할 줄 안다 (I know / I can speak)
იცის = (그/그녀가) 알다 (He/she knows)
