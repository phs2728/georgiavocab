# 챕터 4 — 도시 내 이동 (მგზავრობა ქალაქში)

## 동사 — 가다 (მი-)
წასვლა = 가다 (기본형)
მივდივარ = 나는 간다
მიდიხარ = 너는 간다
მიდის = 그/그녀는 간다
მივდივართ = 우리는 간다
მიდიხართ = 당신(들)은 간다
მიდიან = 그들은 간다

## 동사 — 오다 (მო-)
მოსვლა = 오다 (기본형)
მოვდივარ = 나는 온다
მოდიხარ = 너는 온다
მოდის = 그/그녀는 온다
მოვდივართ = 우리는 온다
მოდიხართ = 당신(들)은 온다
მოდიან = 그들은 온다

## 교통수단
მგზავრობა = 이동, 여행 (travel)
ტაქსი = 택시 (taxi)
ტაქსით = 택시로 (by taxi)
მანქანა = 자동차 (car)
მანქანით = 자동차로 (by car)
ავტობუსი = 버스 (bus)
ავტობუსით = 버스로 (by bus)
ტროლეიბუსი = 트롤리버스 (trolleybus)
ტროლეიბუსით = 트롤리버스로 (by trolleybus)
ტრამვაი = 트램, 전차 (tram)
ტრამვაით = 트램으로 (by tram)
მატარებელი = 기차 (train)
მატარებლით = 기차로 (by train)
ფეხი = 발 (foot)
ფეხით = 걸어서, 도보로 (on foot)

## 장소
ქალაქი = 도시 (city)
ქუჩა = 거리, 길 (street)
სკოლა = 학교 (school)
უნივერსიტეტი = 대학교 (university)
ბიბლიოთეკა = 도서관 (library)
კინო = 영화, 영화관 (cinema)
თეატრი = 극장 (theatre)
ოპერა = 오페라 극장 (opera house)

## 명사·형용사
შეხვედრა = 모임, 회의 (meeting)
ტურისტი = 관광객 (tourist)
მუსიკა = 음악 (music)
ატმოსფერო = 분위기 (atmosphere)
სადილი = 식사, 정찬 (meal / lunch)
სადილზე = 식사에 (at lunch)
საინტერესო = 흥미로운 (interesting)
ლამაზი = 아름다운 (beautiful)
ბევრი = 많은 (many)
ცოტა = 적은, 조금 (few / a little)
ცივი = 추운, 차가운 (cold)
დაკავებული = 바쁜 (busy)

## 부사·시간 표현
ახლა = 지금 (now)
ჯერ = 아직 / 먼저 (yet / first)
ჯერ არა = 아직 아님 (not yet)
კი = 네 / 하지만 (yes / but)
ერთად = 같이, 함께 (together)
ალბათ = 아마도 (probably)
შემდეგ = 그 후에, 그리고 나서 (after / then)
ისე რა = 그저 그래요 (so so)

## 주요 표현
სად მიდიხართ? = 어디 가세요? (Where are you going?)
ერთად წავიდეთ. = 같이 갑시다. (Let's go together.)
ერთი წუთით. = 잠시만요. (Just a moment.)
თუ შეიძლება. = 실례하지만 / 부디. (Please / Excuse me.)
