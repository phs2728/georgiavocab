# 챕터 2 — 사람 이름과 사물

## 의문사
ვინ? = 누구? (Who?)
რა? = 무엇? (What?)
სად? = 어디? (Where?)
საიდან? = 어디에서? (From where?)
სადაური? = 어디 출신의?
რატომ? = 왜? (Why?)
როდის? = 언제? (When?)
რომელი? = 어느 것? (Which?)
როგორ? = 어떻게? (How?)
როგორი? = 어떤? (What kind?)
რამდენი? = 얼마나? / 몇 개? (How many?)

## 사람
ადამიანი = 인간, 사람 (human)
ხალხი = 사람들 (people)
კაცი = 남자, 사람 (man)
ქალი = 여자 (woman)
ბავშვი = 아이 (child)
გოგო = 소녀, 여자아이 (girl)
ბიჭი = 소년, 남자아이 (boy)
მოხუცი = 노인 (old person)
ახალგაზრდა = 젊은이 (young person)
სახელი = 이름 (first name)
გვარი = 성씨 (last name)
და = 자매, 누나/언니/여동생 / 그리고 (and/sister)

## 사물
მაგიდა = 책상, 테이블 (table)
სკამი = 의자 (chair)
კალამი = 펜 (pen)
ჩანთა = 가방 (bag)
წიგნი = 책 (book)
პასპორტი = 여권 (passport)
ვიზა = 비자 (visa)
მანქანა = 자동차 (car)
საგანი = 사물, 물건 (thing)
ჩაი = 차 (tea)
კატა = 고양이 (cat)
ძაღლი = 개 (dog)

## 장소·국가
სახლი = 집 (house)
შინ = 집에 (at home)
ოთახი = 방 (room)
ქალაქი = 도시 (city)
ქვეყანა = 국가, 나라 (country)
საქართველო = 조지아 (Georgia)
თბილისი = 트빌리시 (Tbilisi)
ქუჩა = 거리, 길 (street)

## 소유 대명사
ჩემი = 나의, 내 것 (my/mine)
შენი = 너의, 네 것 (your/yours)
მისი = 그/그녀의 것 (his/hers)
ჩვენი = 우리의, 우리 것 (our/ours)
თქვენი = 당신(들)의 것 (your/yours)
მათი = 그들의 것 (their/theirs)

## 숫자
ნული = 0 (영)
ერთი = 1 (일, 하나)
ორი = 2 (이, 둘)
სამი = 3 (삼, 셋)
ოთხი = 4 (사, 넷)
ხუთი = 5 (오, 다섯)
ექვსი = 6 (육, 여섯)
შვიდი = 7 (칠, 일곱)
რვა = 8 (팔, 여덟)
ცხრა = 9 (구, 아홉)
ათი = 10 (십, 열)
დიდი = 큰 (big)
პატარა = 작은 (little/small)
ბევრი = 많은 (many)
ცოტა = 적은, 조금 (few/a little)

## 시간
დრო = 시간 (time)
დღე = 날, 하루 (day)
გუშინ = 어제 (yesterday)
გუშინწინ = 그저께 (day before yesterday)
გუშინწინისწინ = 3일 전 (three days ago)
დღეს = 오늘 (today)
ხვალ = 내일 (tomorrow)
ზეგ = 모레 (day after tomorrow)
მაზეგ = 글피, 3일 후 (three days from now)
მალე = 곧 (soon)

## 동사 ყოფნა — 현재 (~이다)
ვარ = (나는) ~이다 — 1인칭 단수
ხარ = (너는) ~이다 — 2인칭 단수
არის = (그/그녀는) ~이다 — 3인칭 단수
ვართ = (우리는) ~이다 — 1인칭 복수
ხართ = (당신들은) ~이다 — 2인칭 복수
არიან = (그들은) ~이다 — 3인칭 복수

## 동사 ყოფნა — 과거 (~였다)
ვიყავი = (나는) ~였다 — 1인칭 단수
იყავი = (너는) ~였다 — 2인칭 단수
იყო = (그/그녀는) ~였다 — 3인칭 단수
ვიყავით = (우리는) ~였다 — 1인칭 복수
იყავით = (당신들은) ~였다 — 2인칭 복수
იყვნენ = (그들은) ~였다 — 3인칭 복수

## 동사 ყოფნა — 미래 (~일 것이다)
ვიქნები = (나는) ~일 것이다 — 1인칭 단수
იქნები = (너는) ~일 것이다 — 2인칭 단수
იქნება = (그/그녀는) ~일 것이다 — 3인칭 단수
ვიქნებით = (우리는) ~일 것이다 — 1인칭 복수
იქნებით = (당신들은) ~일 것이다 — 2인칭 복수
იქნებიან = (그들은) ~일 것이다 — 3인칭 복수

## 주요 표현
ჩემი სახელია... = 제 이름은 ...입니다 (My name is...)
დიდი მადლობა = 대단히 감사합니다 (Thank you very much)
არა უშავს = 괜찮습니다 / 상관없습니다 (It's ok)
არა პრობლემა = 문제없습니다 (No problem)
თუ შეიძლება = 실례하지만 / 부탁합니다 (Please)
გაგება = 이해하다 (to understand)
