# 챕터 5 — 식당에서 (რესტორანში)

## 음식·식재료
საჭმელი = 음식, 먹을 것 (food)
სალათა = 샐러드 (salad)
სუპი = 수프 (soup)
კვერცხი = 계란 (egg)
სოკო = 버섯 (mushroom)
პური = 빵 (bread)
კარაქი = 버터 (butter)
ყველი = 치즈 (cheese)
ზეთი = 기름, 오일 (oil)
შაქარი = 설탕 (sugar)

## 음료
სასმელი = 음료, 마실 것 (drink/beverage)
წყალი = 물 (water)
ყავა = 커피 (coffee)
ჩაი = 차 (tea)
რძე = 우유 (milk)
წვენი = 주스 (juice)
ღვინო = 와인 (wine)
ლუდი = 맥주 (beer)

## 식당·식사
საუზმე = 아침 식사 (breakfast)
სადილი = 정찬, 점심·저녁 식사 (meal/lunch)
ვახშამი = 저녁 식사 (dinner)
მიმტანი / ოფიციანტი = 웨이터, 종업원 (waiter)
სტუმარი = 손님 (guest)
ანგარიში = 계산서 (bill)
შეკვეთა = 주문 (order)
მენიუ = 메뉴 (menu)
მაღაზია = 가게, 상점 (shop/store)
ხალხი = 사람들 (people)

## 형용사·대명사
გემრიელი = 맛있는 (delicious)
მოხარშული = 삶은 (boiled/cooked)
ახალი = 새로운, 신선한 (new/fresh)
ძველი = 오래된, 낡은 (old)
კიდევ = 다시, 더 (again/more)
ყველაფერი = 모든 것 (everything)
არაფერი = 아무것도 (~않다) (nothing)
თეთრი = 하얀 (white)
შავი = 검은 (black)
აი = 여기 있습니다 (here it is)

## 동사 — 먹다·마시다·원하다
ჭამს = (그/그녀는) 먹는다 (eats)
სვამს = (그/그녀는) 마신다 (drinks)
მინდა = 나는 원한다 (I want)
გინდა = 너는 원한다 (You want)
უნდა = 그/그녀는 원한다 (He/she wants)

## 격조사 — -ზე (~에, ~위에)
საათზე = ~시에 (at ~ o'clock)
მაგიდაზე = 테이블 위에 (on the table)
შეხვედრაზე = 회의에 (at the meeting)
სადილზე = 식사에 (at lunch)

## 장소 파생어 (სა—ო)
სამზარეულო = 부엌, 주방 (kitchen) ← მზარეული (요리사)
სასტუმრო = 호텔 (hotel) ← სტუმარი (손님)
საელჩო = 대사관 (embassy) ← ელჩი (대사)
საქართველო = 조지아 (Georgia) ← ქართველი (조지아 사람)
საინტერესო = 흥미로운 (interesting) ← ინტერესი (관심)

## 식당 표현
რა გნებავთ? = 무엇을 원하십니까? / 주문하시겠습니까?
ინებეთ! = 여기 있습니다!
გემრიელად მიირთვით! = 맛있게 드세요!
მიირთვით! = 드세요! / 마음껏 드세요!
მობრძანდით! = 어서 오세요! / 들어오세요! (정중형)
დაბრძანდით! = 앉으세요! (정중형)
მიბრძანდით! = 먼저 가세요! / 먼저 하세요! (정중형)
კარგად ბრძანდებოდეთ! = 안녕히 가세요/계세요! (정중형)
